\subsection{Detailed Description}
A 3D drawable point. 

A 3D point that can be drawn in a view. Note that this is different from a \hyperlink{classGKPoint}{G\+K\+Point}, this one is a full \hyperlink{classGKObject}{G\+K\+Object} with support for extensible data (new attributes), editors, drawers... The \hyperlink{classGKPoint}{G\+K\+Point} is the generic definition of a point (the coordinates).

\subsection{Detailed Description}
A 2D Circle. 

A \hyperlink{classGKCircle}{G\+K\+Circle} represents a circle with a radius (expressed in metres), and is drawn with a fill and an outline.
\begin{DoxyItemize}
\item The outline is drawn using the pen\+Style, pen\+Color, pen\+Size and pen\+Size\+Units properties.
\item The fill is drawn using the brush\+Style and brush\+Color properties. 
\end{DoxyItemize}

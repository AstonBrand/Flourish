Detailed Description

Represents a connected vehicle (e.g., a mobile station) in the framework.

Copyright TSS 2017

Author: Natale Patriciello natal.nosp@m.e.pa.nosp@m.trici.nosp@m.ello.nosp@m.@aims.nosp@m.un.c.nosp@m.om

The vehicle position is given by the Aimsun simulator. In this framework, it is managed the connection (and the disconnection) of the vehicle with others vehicles/infrastructures. This is done in a very transparent way, equipping the vehicle with an access point, that works like the V2XSimpleAP object.

The only difference is that it should maintain a list of the in-range vehicles and stations, because after each simulation step it is required to update the active links through a call to the method updateLinks.

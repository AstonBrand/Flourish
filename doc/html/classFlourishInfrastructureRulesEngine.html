Detailed Description

Takes decision for an infrastructure giving the received messages.

Copyright TSS 2017

Author: Natale Patriciello natal.nosp@m.e.pa.nosp@m.trici.nosp@m.ello.nosp@m.@aims.nosp@m.un.c.nosp@m.om

Takes decisions based on the incoming messages. The 'evaluate' methods, are called when a message should be evaluated (please note that they are empty now and should be implemented).
